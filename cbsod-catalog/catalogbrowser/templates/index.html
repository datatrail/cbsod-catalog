Demo CBS Open Data Services using Python - Catalog browser


StatLine is the electronic databank of CBS (Statistics Netherlands). It enables users to compile their own tables and graphs. The information can be accessed, printed and downloaded free of charge.

StatLine is made available as Open Data via the ODATA protocol. CBS published a tutorial describing the Data Services provided and a tutorial how to retrieve data using PowerPivot for Excel.

The Data Services can also be consumed by other programs or languages. This demo shows how Python can be used to replicate (a simplified version of) the CBS Open Data portal. Users can browse the catalog, select a table and view the content of the table.

The application was build using Flask and py2cbs, a small client library for working with CBS (Netherlands Statistics) Open Data from within Python applications and from the command line.
